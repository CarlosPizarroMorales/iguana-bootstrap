## Links

La notación básica para crear links consiste en utilizar un par de corchetes seguido de un paréntesis.[texto-del-link](url).

Entonces enlace a [Google](https://google.com "texto-alternativo"). Variantes del enlace: 

1. Cuando no quiero que mi texto se llene de links, enlaces a referencias: [Google][01]
2. Cuando quiero agregar una imagen es la notación básica con un ! delante del corchete: ![Goku](https://www.hdwallpaper.nu/wp-content/uploads/2015/06/Dragon-Ball-Z-Wallpapers-HD.jpg)
3. Enlace a una imagen local ![Iguana](./assets/img/iguana-1.jpg) 
4. Enlace interno a id [Ir al principio](#proyecto-iguana)
5. Enlace a línea de mi código en el proyecto. Es la sintaxis básica más el # y luego Ln donde n es el número de la línea. [codigo-feo-aca](./index.html#L30)
